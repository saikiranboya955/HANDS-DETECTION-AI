✋ Hand Detection AI – Neon Aura AR
A real-time AI-powered hand tracking system that transforms human gestures into an interactive neon visual and audio experience. Built using MediaPipe Hands, this project detects hand landmarks and interprets gestures to create immersive AR effects.

🌟 Features

.🔍 Real-time Hand Detection
Detects up to 2 hands simultaneously using advanced ML models.
.🧠 Gesture Recognition
Pinch detection (Thumb + Index finger)
Open hand / Fist classification
Finger spread analysis
.🎨 Dynamic Visual Effects
Neon glowing fingertips
Particle trails & shockwaves
Interactive lightning between hands
Matrix-style animated background
.🎧 Audio Interaction
Sound effects triggered by gestures
Dynamic hum based on hand distance
.🎭 Theme Engine
Switch between visual themes like:
Rainbow 🌈
Cyberpunk ⚡
Lava 🔥
Ocean 🌊
Galaxy 🌌
